# الأهرام للصناعات والتشكيلات المعدنية | Al-Ahram for Metal Industries

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![PWA](https://img.shields.io/badge/PWA-Ready-orange.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

موقع تعريفي احترافي لشركة **الأهرام للصناعات والتشكيلات المعدنية**، رائدة أعمال تشكيل المعادن واللحام في اليمن. يتميز الموقع بتصميم عصري وأداء عالٍ مع دعم كامل لتقنيات الويب المتقدمة (PWA).

A professional informational website for **Al-Ahram for Metal Industries**, a leader in metal forming and welding in Yemen. The site features a modern design, high performance, and full Progressive Web App (PWA) support.

---

## 🌟 المميزات | Features

- 📱 **تصميم مستجيب بالكامل (Responsive Design):** يتوافق مع جميع أحجام الشاشات (هواتف، تابلت، حواسب).
- 🚀 **تطبيق ويب تقدمي (PWA):** إمكانية تثبيت الموقع كشريحة على الهاتف والعمل في وضع الأوفلاين.
- 🎨 **هوية بصرية قوية:** استخدام نظام ألوان احترافي (الذهبي والداكن) يعكس طابع الصناعة.
- 🛠️ **عرض الخدمات:** تفصيل شامل لخدمات اللحام (TIG/MIG)، القص بالليزر، وخراطة CNC.
- 📞 **سهولة التواصل:** تكامل مباشر مع WhatsApp ونموذج اتصال احترافي.
- ⚡ **أداء عالي:** سرعة تحميل فائقة وتحسين لمحركات البحث (SEO).

---

## 🛠️ التقنيات المستخدمة | Technologies Used

- **Frontend:** HTML5, CSS3 (Vanilla CSS), JavaScript (Vanilla JS).
- **Styling:** CSS Grid, Flexbox, Glassmorphism, CSS Animations.
- **PWA Tools:** Service Workers, Web App Manifest.
- **Icons & Fonts:** FontAwesome 6.4, Google Fonts (Cairo).
- **Deployment:** Ready for Netlify/Vercel (includes `netlify.toml`).

## 📂 هيكلة المشروع | Project Structure

```text
├── css/              # ملفات التنسيق (Stylesheets)
├── js/               # ملفات الجافاسكريبت (Scripts)
├── images/           # الصور العامة والأيقونات
├── images3/          # صور المنتجات والخدمات (AI Generated)
├── icons/            # أيقونات التطبيق (PWA Icons)
├── index.html        # الصفحة الرئيسية
├── about.html        # صفحة من نحن
├── services.html     # صفحة الخدمات
├── contact.html      # صفحة اتصل بنا
├── manifest.json     # ملف تعريف الـ PWA
└── sw.js             # ملف الـ Service Worker
```
